# User Settings

Access your settings from the navigation menu at any time. Changes take effect immediately and sync across devices.

## Profile

Update your **bio** and **avatar** from the Profile section. Your public profile page is linked from here.

## Appearance

Choose a display theme:

- **Light** — Default light mode
- **Dark** — Dark mode
- **High Contrast** — Light mode with increased contrast
- **High Contrast Dark** — Dark mode with increased contrast
- **Colorblind-Friendly** — Adjusted color palette
- **System** — Follows your device's system preference

## Notifications

Connect your Discord account to receive game notifications as Discord DMs.

**Connecting Discord** — Click **Connect Discord** to link your account via Discord OAuth. Once connected, your Discord username is shown and a **Disconnect** button lets you unlink at any time.

**Per-event toggles** — After connecting, you can enable or disable Discord DMs for each notification type individually:

- *Direct events* (on by default): Private messages, action results, character approved, application approved, new handouts, common room posts
- *Game activity* (off by default): Comment replies, character mentions, action submitted, phase created, game state changes, application submitted

Changes take effect immediately for future notifications.

> Your Discord account must share a server with the ActionPhase bot for DMs to work.

## Reading

The **Comment Read Tracking** setting controls how new comments are highlighted when you return to a page.

- **Automatic** — Comments added since your last visit are highlighted automatically.
- **Manual** — You mark individual comments as read yourself. Read comments fade out.
